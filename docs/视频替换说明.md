# 视频替换说明

当前网站已经把 20 个动作绑定到固定视频文件名。以后想换某个动作的视频时，最简单的方式是：

1. 准备一个新的 `.mp4` 视频。
2. 把新视频改成下面表格里的同一个文件名。
3. 覆盖放到 `client/public/exercise-videos/` 目录。
4. 重新提交并推送到 GitHub，GitHub Pages 会自动更新。

只要文件名不变，动作数据里的 `videoUrl` 就不用改。

## 视频文件要求

- 单个视频推荐 1-3MB。
- 单个视频尽量不要超过 5MB。
- 格式优先使用 `.mp4`，编码建议 H.264。
- 分辨率建议 480p 或 720p。
- 时长建议 5-10 秒，能看清完整动作即可。

## 注意

如果你想换成完全不同的文件名，也可以，但需要同时修改 `server/src/db.js` 里的 `exerciseVideoMetadata` 和重新导出 `client/public/data/catalog.json`。

如果只是替换视频内容，保持文件名不变是最省事的方式。
