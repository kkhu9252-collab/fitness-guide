# 图片替换说明

当前网站的动作封面已经改为固定本地图片文件。以后想替换某个动作的封面时，最简单的方式是：

1. 准备新的器材图片，建议 JPG 格式。
2. 把新图片改成下面表格里的同一个文件名。
3. 覆盖放到 `client/public/equipment-covers/` 目录。
4. 重新提交并推送到 GitHub，GitHub Pages 会自动更新。

只要文件名不变，动作数据里的 `imageUrl` 就不用改。

## 图片文件要求

- 图片里尽量只出现器材，不要人物。
- 优先使用白底或浅色干净背景。
- 建议保存为 `.jpg`。
- 推荐尺寸为正方形，例如 1024 x 1024。
- 单张图片建议控制在 500KB 以内。

## 注意

如果只是替换图片内容，保持文件名不变是最省事的方式。

如果你想换成完全不同的文件名，需要同时修改 `server/src/db.js` 里的 `exerciseImageUrls`，并重新导出 `client/public/data/catalog.json`。
